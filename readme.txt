1. download and install xampp.
2. make sure your php virsion higher then 7.3.0.
3. put 'wedding' folder inside your c:/xampp/htdocs.
4. run xampp and activate the Apache server and mySQL database.
5. create wedding_db on http://localhost/phpmyadmin/.
6. import table from sql file (wedding_db.sql).
7. run url http://localhost/wedding/public/.
